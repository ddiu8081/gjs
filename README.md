# gjs
关雎说
Gjs is a birthday gift for gjs.

# 特性
* 每天更新一句话，0点自动切换下一条
* 每天一个主题颜色
* 可设置隐藏语句，点击才会显示隐藏内容
* 图片、音乐、视频、链接分享
* 历史记录查看（上一页/下一页按钮）
* 自适应页面
* 点赞功能（已下线）
* 后台发布（暂未实现）
* ...
